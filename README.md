## Requirements
* Echll (≥ 0.0)
* boost (≥ 1.50)
* boost::mpi (≥ 1.50)
* boost::serialization (≥ 1.50)
* cmake (≥ 2.8.7)
* openmpi (≥ 1.6)
* c++ compiler (gcc ≥ 4.8, clang ≥ 3.3, intel icc (≥ 11.0).
